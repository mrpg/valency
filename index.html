    Welcome to Valency

    Valency is a programming language designed for simplicity, code readability and consistency. It follows the paradigm that everything is made of variables. Functions, strings and numbers of any kind - these are all variables. Valency is a very dynamic programming language that allows nearly everything theoretically possible.

    The Valency reference implementation - the interpreter - aims to be easy to understand and yet powerful. Try it yourself!
